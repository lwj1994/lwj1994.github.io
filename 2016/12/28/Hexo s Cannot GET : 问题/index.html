Hexo s Cannot GET / 问题

2016.12.28 luwenjie hexo  热度 ℃

在 Mac 上搭建 hexo 时，执行到 hexo s 后，访问localhost:4000 出现

Cannot GET /

解决方法

hexo init 之后需要执行 npm intall

完整步骤示例

npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server
