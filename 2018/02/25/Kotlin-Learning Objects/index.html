Companion Objects 伴随对象

在一个类内部声明的对象可以用 companion 关键字标记。表示这个类的伴随对象。

调用的时候直接通过 MyClass.create() 调用对象内部的方法。 其实调用的是 MyClass.Factory.create()。

类似于 Java 的静态方法直接通过类名调用。
每个类只能有一个伴随对象。所以伴随对象的名字可以省略不写，内部会使用 Companion 代替名字。

此时实际调用的就是 MyClass.Companion.create()

注意，虽然伴随对象看起来像 Java 的静态成员，但是在运行时它们仍然是对象的实例。

对象表达式和对象声明的语义区别

  • 对象表达式被执行后立即执行初始化，对象声明是懒加载的。
  • 一个类被类加载器加载时，先初始化内部伴随对象，再初始化类。
